ÇEREZ POLİTİKASI

Çerezleri nasıl kullandığımızı öğrenerek gezinme deneyiminizi nasıl geliştirdiğimizi keşfedin.

Çerez Nedir?

Çerezler, bir web sitesini ziyaret ettiğinizde cihazınıza kaydedilen küçük metin dosyalarıdır. Web sitesinin düzgün çalışmasını sağlamakla birlikte, tercihleriniz ve etkileşimlerinizi hatırlayarak kullanıcı deneyiminizi iyileştirmeye yardımcı olurlar.

Çerezler, web deneyiminizi daha sorunsuz ve kişiselleştirilmiş hale getirmek için web tarayıcınız tarafından depolanan küçük veri parçalarıdır.

Çerezler web deneyiminizi daha sorunsuz ve kişisel hale getirir.

Çerezleri Nasıl Kullanıyoruz?

KaderDüello olarak çerezleri aşağıdaki amaçlarla kullanıyoruz:

İşlevsellik

Web sitesinin işlevselliğini artırmak.

Analiz

Site trafiğini analiz etmek.

Kişiselleştirme

Kullanıcı deneyiminizi kişiselleştirmek.

Kullandığımız Çerez Türleri

Zorunlu Çerezler

Her zaman aktif

Bu çerezler, web sitesinin temel işlevleri için gereklidir ve sitenin düzgün çalışmasını sağlar.

Performans Çerezleri

İsteğe bağlı

Ziyaretçilerin web sitemizle nasıl etkileşimde bulunduğunu anlamamıza yardımcı olmak için anonim veriler toplar.

İşlevsellik Çerezleri

İsteğe bağlı

Tercihlerinizi hatırlayarak size daha kişiselleştirilmiş bir deneyim sunar.

Reklam Çerezleri

İsteğe bağlı

İlgi alanlarınıza uygun reklamlar göstermek için kullanılır.

Ayrıca, üçüncü taraf hizmet sağlayıcılarına (örneğin analiz şirketleri ve reklam verenler) çerezleri yerleştirme izni verebiliriz. Bu çerezler, hizmetlerimizi iyileştirmek ve size özel reklamlar sunmak için kullanılır.

Çerezleri Yönetme

Çerezleri kontrol etme veya devre dışı bırakma hakkına sahipsiniz. Ancak çerezleri devre dışı bırakmak, web sitesinin bazı işlevlerini etkileyebilir.

Çerezleri yönetmek için tarayıcınızın ayarlarını kullanabilirsiniz:

Google Chrome: Menü → Ayarlar → Gizlilik ve güvenlik → Çerezler ve diğer site verileri

Mozilla Firefox: Menü → Seçenekler → Gizlilik ve güvenlik → Çerezler ve site verileri

Safari: Tercihler → Gizlilik → Çerezler ve web sitesi verileri

Microsoft Edge: Ayarlar → Çerezler ve site izinleri → Çerezleri yönet ve sil

Unutmayın:

Çerezleri devre dışı bıraksanız bile, temel işlevselliği sağlamak için bazı zorunlu çerezler aktif kalmaya devam edebilir.

Çerezlerin Faydaları

Çerezleri kullanarak size sunduğumuz faydalardan bazıları şunlardır:

Hızlı Giriş

Giriş bilgilerinizi hatırlayarak daha hızlı oturum açmanızı sağlar.

Kişiselleştirilmiş Deneyim

Tercihlerinizi hatırlayarak size özel bir deneyim sunar.

İyileştirilmiş Performans

Web sitesinin daha hızlı ve verimli çalışmasına yardımcı olur.

Daha İyi Hizmet

Kullanım verilerini analiz ederek hizmetlerimizi geliştirir.
